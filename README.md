## ✨ Features

### 🎮 For Kids
| | Feature | Description |
|---|---|---|
| 🧩 | **Multiple Choice** | Tap the right answer from big, friendly buttons |
| 👀 | **Visual Counting** | See real objects — count cars, animals, or flowers |
| ⭐ | **Stars & Rewards** | Earn stars and watch celebration animations |
| 🎵 | **Sound Effects** | Happy sounds for correct answers, gentle encouragement for retries |
| 🌈 | **Colorful Themes** | Switch between Cars, Animals, and Plants anytime |
| 📏 | **Just Right** | Short 2–5 minute sessions perfect for little attention spans |

### 👨‍👩‍👧 For Parents
| | Feature | Description |
|---|---|---|
| 🔒 | **Parent Gate** | Hold-to-unlock keeps settings safe from tiny fingers |
| 📊 | **Progress Tracking** | See accuracy, streaks, and stars earned per skill |
| ⚙️ | **Full Control** | Set difficulty, enable/disable operations, adjust session length |
| 🔇 | **Sound Controls** | Toggle sound effects and music on/off |
| 🎯 | **Daily Goals** | Set a daily problem target to build good habits |
| 🚫 | **No Ads, No Accounts** | 100% private, local-first, kid-safe |

## 🛠️ Getting Started

### Prerequisites

- 🍎 **macOS** with Xcode 15.4+
- 📱 **iOS 17.0+** target device or simulator

### Build & Run

```bash
# Clone the repo
git clone https://github.com/frankcheng-coder/math_buddy.git

# Open in Xcode
cd math_buddy
open KidsMathApp.xcodeproj

# Select a simulator and hit ▶️ Run!
```

## 📁 Project Structure

```
🗂️ KidsMathApp/
├── 🚀 App/              Entry point, state management, routing
├── 📦 Models/            Theme, MathOperation, DifficultyLevel, Progress
├── 🧠 ViewModels/        Business logic for each screen
├── 🎨 Views/
│   ├── 🏠 Home/          Main menu with operation cards
│   ├── 🎨 ThemeSelection/ Pick your favorite theme
│   ├── 🎮 Practice/      The math gameplay screen
│   ├── 👨‍👩‍👧 Parent/        Settings behind parent gate
│   ├── 📊 Progress/      Stars, streaks, and skill breakdown
│   └── 🧩 Shared/        Reusable buttons, animations, components
├── 🎲 Games/             Math engines (Add, Subtract, Multiply, Divide)
├── 🎭 Themes/            Theme manager and theme definitions
├── ⚙️ Services/          Audio, rewards, and persistence
└── 📂 Resources/         Assets and sounds
```
